`/Users/apple/program/CCC/scripts/ccc-exec-commit.sh` L82-100 的 Python heredoc。

把:
```python

    data = json.load(f)
```

改成(兼容 JSON 文档 + JSONL 两种格式):
```python
with open(fp) as f:
    content = f.read().strip()
if not content:
    data = {}
elif content.startswith('['):
    data = {"phases": json.loads(content)}
elif '\n' in content and not (content.startswith('{') and content.endswith('}')):
    phases = [json.loads(line) for line in content.splitlines() if line.strip()]
    data = {"phases": phases}
else:
    data = json.loads(content)
```

后面用 `data.get('phases', [data])` 兼容两种 schema。
