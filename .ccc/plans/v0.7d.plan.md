## 改动 1：重建 4 窗口

### 做什么
当前窗口 0/1/2/3/4 的 cwd 混乱（之前 send-keys 弄混了）。杀掉全部，按顺序重建。

### 怎么做
```bash
tmux kill-session -t claude-code  # 杀掉整个 session

# 新建
tmux new-session -d -s claude-code -n ccc -c /Users/apple/program/CCC
tmux new-window -t claude-code -n qb -c /Users/apple/program/projects/qb
tmux new-window -t claude-code -n hp -c /Users/apple/program/hp
tmux new-window -t claude-code -n xianyu -c /Users/apple/program/xianyu
tmux new-window -t claude-code -n monitor -c /Users/apple/program/CCC \
  'bash -c "while true; do clear; echo \"=== \$(date) monitor ===\"; tmux list-windows -t claude-code; echo; for w in \$(tmux list-windows -t claude-code -F \"#I\"); do echo \"--- window \$w ---\"; tmux capture-pane -t claude-code:\$w -p 2>/dev/null | tail -5; done; sleep 10; done"'
```
顺序：0=ccc, 1=qb, 2=hp, 3=xianyu, 4=monitor。

- `tmux list-windows -t claude-code` 输出 5 窗口顺序正确
- `tmux send-keys ... pwd` 每个窗口返回正确路径
