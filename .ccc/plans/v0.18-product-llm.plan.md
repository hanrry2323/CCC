# v0.18-product-llm

> 标题: product 改 Claude API 拆任务（不是 opencode）
> 创建: 2026-07-07

## 目标

product_role() 当前只写模板（"待细化" + "由 dev 拆"），没有真拆任务。改为调 Claude API 写 SPEC-合规 plan。

## 实施

1. `ccc-board.py` product_role() 改：
   - 不再扫描 backlog 自动移动（已改）
   - 改为调 Claude API：`client.messages.create(model="claude-sonnet-4-20250514", max_tokens=4096)`
   - prompt 传入 task 内容 + AGENTS.md + 历次 plan 作为参考
   - Claude 返回 plan.md 内容 + phases.json timeout
   - 写 plan.md + phases.json
   - 挪 backlog → planned（手动调用时）

2. product_role 新增 `--promote <task_id>` 参数：
   - 只处理指定 task（不扫全部）
   - 调 Claude API → 写 plan/phases → 挪 planned

3. 依赖：
   - `pip install anthropic` 或调 claude CLI
   - API key 从环境变量 ANTHROPIC_API_KEY

## 验收

- backlog 放一个测试任务 → `python3 scripts/ccc-board.py product --promote test-task` → planned 出现 test-task.plan.md
- plan.md 包含 SPEC 合规验收项（每条可程序化验证）
- phases.json 包含 timeout
