## Phase 3: Verdict 门禁

### 目标

当前 FAIL/FALLBACK verdict 写入后，commit 已入库，无回滚机制。
改为：Engine 检测到 FAIL/FALLBACK → `git revert` → 退回 `planned`。

### 改动清单

#### 3.1 新增 `_revert_task_commit()` 函数

**文件**：`scripts/ccc-engine.py`

**位置**：在 `_verdict_is_timeout()` 函数（约 L447-454）之后、`_verdict_file()` 附近插入。

新增以下函数体：

```python
def _revert_task_commit(ws: Path, tid: str) -> bool:
    """Verdict FAIL 时回滚 task 的最后一个 commit。

    读取 phases.json 拿到 commit hash → git revert → 退回 planned。
    失败仅日志记录，不阻断 engine 主循环。

    Returns:
        True = 回滚成功；False = 失败
    """
    phases_file = ws / ".ccc" / "phases" / f"{tid}.phases.json"
    if not phases_file.exists():
        return False
    try:
        phases_data = _load_phases(tid, ws)
        commits = [
            p.get("commit", "") for p in phases_data
            if p.get("commit") and p.get("commit") not in ("null", "None", "")
        ]
        if not commits:
            return False
        last_commit = commits[-1]
    except (OSError, json.JSONDecodeError) as exc:
        engine_log(f"[verdict-gate] {tid} 读 commit hash 失败: {exc}")
        return False

    try:
        result = subprocess.run(
            ["git", "revert", "--no-edit", last_commit],
            capture_output=True, text=True, timeout=30,
            cwd=str(ws), env=_sanitized_env(),
        )
        if result.returncode != 0:
            engine_log(
                f"[verdict-gate] {tid} revert {last_commit[:12]} 失败: "
                f"{result.stderr[:200]}"
            )
            return False
        engine_log(
            f"[verdict-gate] {tid} 已 revert commit {last_commit[:12]}"
        )
        return True
    except Exception as exc:
        engine_log(f"[verdict-gate] {tid} revert 异常: {exc}")
        return False
```

**注意**：
- `subprocess` 在文件顶部已导入，不需要重新 import
- `json` 在文件顶部已导入
- `_sanitized_env()` 在文件顶部已导入（line 34）

```python
def _verdict_file(ws: Path, tid: str) -> Path:
    return ws / ".ccc" / "verdicts" / f"{tid}.verdict.md"

def _verdict_is_valid(ws: Path, tid: str) -> bool:
    vf = _verdict_file(ws, tid)
    return vf.is_file() and vf.stat().st_size > 0

def _clear_verdict(ws: Path, tid: str) -> None:
    vf = _verdict_file(ws, tid)
    try:
        vf.unlink()
    except OSError:
        pass
```
