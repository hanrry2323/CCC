# Release v0.42.1 — CCC Hub Loop Engineer

**Tag:** `v0.42.1` · **Date:** 2026-07-17

## Why CCC

**CCC（Connect–Claude Code）** is a **Loop Engineer**: you set intent in seconds; the system orchestrates, executes, verifies, retries, and can evolve.

- **CCC Hub** is the product entry (chat + board + console) — not a third-party agent IDE shell  
- **Task → tool routing → Skill + Prompt = role** (unlimited roles; no role supermarket)  
- Users do **not** pick roles or memorize skills

## Quick start

```bash
git clone https://github.com/hanrry2323/CCC.git && cd CCC
bash scripts/install-board-plist.sh --start
bash scripts/install-hub-plist.sh --start
open http://127.0.0.1:7777   # ccc / ccc
```

Then in Hub: **对齐基线 → 定稿方案 → 转任务 → 下达并开工**.  
Enable the queue consumer only when you want automation:

```bash
bash scripts/ccc-autostart-guard.sh enable --start
```

See [`docs/GETTING-STARTED.md`](../GETTING-STARTED.md).

## What's new in v0.42.1

- **iOS viewport lock** (no full-page scroll to reach the composer)  

### Pipeline bridge

- Wake Engine + workspace registry on dispatch  
- Adopt existing plans / backfill empty scope  
- Safer engine start via `may_start_engine`
