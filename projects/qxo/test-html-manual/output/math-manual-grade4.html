第三章

课程标准与核心素养

基于《义务教育数学课程标准（2022 年版）》，明确四年级数学的核心素养与课程目标。

一、数学核心素养（2022 课标）

根据 2022 版课标，小学数学核心素养主要包括以下三个方面，贯穿四年级教学全过程：

  • 会用数学的眼光观察现实世界：能从生活情境中抽象出数量关系与空间形式
  • 会用数学的思维思考现实世界：能运用归纳、演绎、类比等推理方式分析问题
  • 会用数学的语言表达现实世界：能用数学符号、图表、模型等表达和交流

二、第二学段（4-6 年级）课程目标

四年级属于第二学段，本学段在"数与代数 / 图形与几何 / 统计与概率 / 综合与实践"四大领域有如下总体要求：

  • 数与代数：扩展对自然数的认识，理解大数的意义；掌握三位数乘两位数、除数是两位数的除法；初步认识分数与含义；探索运算规律
  • 图形与几何：认识线段、直线、射线与角；理解平行与垂直；认识三角形、平行四边形、梯形的特征；体会图形的运动
  • 统计与概率：经历简单的数据收集、整理、描述和分析过程；认识条形统计图与折线统计图
  • 综合与实践：在真实情境中综合运用所学知识解决问题，体会数学与生活的联系

三、四年级教学重点

四年级是小学数学从"具体运算"向"抽象运算"过渡的关键年级，教学应重点关注：

  1. 从算术思维向代数思维的早期渗透（用字母表示数、等量关系）
  2. 空间观念的发展（从立体到平面、从直观到抽象）
  3. 数据处理意识的建立（从单一数据到多源数据的分析）
  4. 数学应用意识的培养（综合与实践领域的真实问题解决）
第四章

上册知识点详解

人教版四年级上册，按四大领域组织。每个知识单元含学习目标 / 知识要点 / 教学难点 / 易错点 / 教学建议五要素。

领域一 · 数与代数

4.1.1 大数的认识
学习目标
认识万级和亿级的数位、计数单位；掌握亿以内数的读法和写法；会用"四舍五入"法求近似数；体会大数在生活中的应用。
知识要点
数位顺序表（个位、十位…千亿位）；计数单位（一、十…千亿）；每四位一级（个级、万级、亿级）；多位数读写规则；用"万"或"亿"作单位表示大数。
教学难点
中间或末尾有 0 的多位数读法（如 20004000 读作"二千万四千"）；数位与计数单位的区分；亿以内数的写法（尤其 0 的占位）。
易错点
①读数时漏读"零"或多读"零"；②写数时数位对错；③"四舍五入"看错省略位；④把"="与"≈"用混。
教学建议
①用数位顺序表贯穿全单元，让学生随时对照；②结合成都本地数据（如人口、GDP）增强情境感；③设计"读—写—比"三步训练，每堂课 5 分钟巩固；④"四舍五入"借助数轴直观理解，避免死记口诀。
4.1.2 三位数乘两位数
学习目标
掌握三位数乘两位数的笔算方法；理解积的变化规律；能用所学知识解决简单的实际问题。
知识要点
三位数乘两位数的笔算步骤；因数中间或末尾有 0 的简算；积的变化规律（一个因数不变，另一个因数乘几，积也乘几）；常见数量关系：单价×数量=总价、速度×时间=路程。
教学难点
竖式书写时部分积的对位；积的变化规律的归纳与表述；进位时的连续处理。
易错点
①部分积的数位对错；②进位漏加；③末尾 0 的简算时少补 0；④"速度×时间=路程"中单位不统一。
教学建议
①先复习两位数乘两位数，迁移算法；②用方格图/面积模型直观解释部分积的含义；③对比"末尾有 0"和"中间有 0"两种情况，避免混淆；④结合生活实例（出行路程、采购总价）巩固数量关系。
4.1.3 除数是两位数的除法
学习目标
掌握除数是两位数的笔算除法；理解商的变化规律；能用四舍五入法试商、调商；解决简单实际问题。
知识要点
四舍五入法试商；调商的方法（四舍试商可能偏大需调小，五入试商可能偏小需调大）；商的变化规律（被除数和除数同时乘/除以相同的数，商不变）；有余数除法的验算。
教学难点
试商方法的灵活选择；调商时的判断；商不变规律中"余数变化"的理解。
易错点
①试商后忘记乘减直接写商；②调商后未重新计算；③商的数位定位错；④"商不变"中余数也认为不变。
教学建议
①设计"试商—调商"专项训练，每天 3 题；②用"估商—验商—定商"三步法口诀；③商不变规律用具体例题（24÷6、48÷12、96÷24）让学生观察归纳；④结合购物分摊、平均分配等情境出题。

领域二 · 图形与几何

4.2.1 线段、直线、射线与角
学习目标
认识线段、直线、射线的特征与区别；认识角的定义、表示方法；会用量角器量角、画指定度数的角。
知识要点
线段（两个端点、可度量）、直线（无端点、不可度量）、射线（一个端点、向一方无限延伸）；角的定义（从一点引出两条射线所组成的图形）；角的表示（∠1、∠ABC）；角的分类（锐角、直角、钝角、平角、周角）。
教学难点
三种线的区别与联系；射线方向性的理解；量角器的正确使用（中心对顶点、0 刻度线对一边）。
易错点
①直线与射线混淆；②量角时读错内/外圈刻度；③画角时方向错；④"平角 = 180°"与"直线"概念混淆。
教学建议
①用激光笔/手电筒演示射线；②量角器教学分两步：先对点对边、再读刻度；③画角采用"一对点、二对线、三找点、四连线"四步法；④结合折纸活动体会平角与周角。
4.2.2 平行与垂直
学习目标
理解同一平面内两条直线的位置关系（平行、相交、垂直）；会用直尺和三角尺画平行线与垂线；理解平行四边形和梯形的高。
知识要点
平行的定义（在同一平面内不相交）；垂直的定义（相交成直角）；平行与垂直的符号表示（∥、⊥）；点到直线的距离（垂线段最短）；画垂线和平行线的方法。
教学难点
"同一平面内"这一前提的理解；用三角尺画平行线/垂线的操作步骤；点到直线的距离与垂线段的关系。
易错点
①忽视"同一平面内"；②画垂线时三角尺放错位置；③把"垂直"等同于"竖直"；④平行四边形的高画到了顶点上。
教学建议
①用方格纸辅助理解平行；②"画平行线"用"平移三角尺"操作法，分步示范；③结合生活中的平行与垂直（斑马线、门框）增强直观；④画高时强调"从底边的对边任意一点向底边作垂线"。
4.2.3 平行四边形与梯形
学习目标
认识平行四边形和梯形的特征；理解它们与长方形、正方形的关系；会画平行四边形和梯形的高。
知识要点
平行四边形（两组对边分别平行）；梯形（只有一组对边平行）；平行四边形的底与高；梯形的上底、下底、高、腰；等腰梯形；四边形分类关系图（正方形 ⊂ 长方形 ⊂ 平行四边形 ⊂ 四边形）。
教学难点
四边形之间的包含关系；平行四边形"高"的多对应性（任一边都可作底）；梯形高的画法。
易错点
①梯形与平行四边形混淆（"只有一组"vs"两组"）；②高的画法错位；③认为"长方形不是平行四边形"；④梯形上底/下底颠倒。
教学建议
①用集合图直观展示四边形关系；②用四根木条做"平行四边形变形"演示，体会不稳定性；③"高"的教学采用"哪条边作底，就从对边向它作垂线"的统一表述；④结合成都街区的梯形屋顶、平行四边形地砖等实例。

领域三 · 统计与概率

4.3.1 条形统计图
学习目标
认识条形统计图（1 格表示多个单位）；能根据数据完成条形统计图；能从图中获取信息、分析数据。
知识要点
条形统计图的结构（标题、横轴、纵轴、单位、直条）；1 格表示多个单位的确定方法；制作条形统计图的步骤；从图中读数据、比较数据、做出简单判断。
教学难点
1 格表示多少单位的确定（依据数据范围与美观）；不完整数据的标注；分析数据时从"读数"到"判断"的过渡。
易错点
①1 格表示的单位错；②纵轴刻度不连续时未标"折线"；③直条高度对错刻度；④只读数不分析。
教学建议
①用班级真实数据（如身高、阅读量）作图；②"1 格表示几"让学生讨论决定，培养数据直觉；③分析环节用"我看到… 我想到… 我建议…"三步法；④对比表格与统计图，体会图形直观性。

领域四 · 综合与实践

4.4.1 1 亿有多大
学习目标
通过探究活动体会 1 亿的大小；经历"猜想—设计—测量—推算—得出结论"的研究过程；培养数感与估测能力。
知识要点
1 亿的数感建立（1 亿粒大米约多少千克？1 亿步约多少千米？）；推算方法（局部测量 → 按比例放大）；活动方案的设计与实施。
教学难点
由"小样本"推算"1 亿"的逻辑；活动方案的设计；估测结果的合理性判断。
易错点
①测量样本太小，误差过大；②推算时单位换算错；③结论脱离常识（如 1 亿粒米 > 一座山）；④活动记录不完整。
教学建议
①分组探究不同主题（大米、纸张、步数、硬币），增加多样性；②强调"先估、再测、后算"三步；③用成都到北京的距离（约 1800 km）作参照，让结果具象化；④活动后做"我的发现"分享。
第五章

下册知识点详解

人教版四年级下册，结构同上册。

领域一 · 数与代数

5.1.1 四则运算的意义与关系
学习目标
理解加、减、乘、除四则运算的意义与各部分关系；掌握 0 在四则运算中的特性；能正确进行四则混合运算。
知识要点
加减关系（和—加数=另一加数；被减数—减数=差）；乘除关系（积÷因数=另一因数；被除数÷商=除数）；0 的运算特性（a+0=a、a−0=a、a×0=0、0÷a=0 (a≠0)）；四则混合运算顺序。
教学难点
0 不能作除数的理解；混合运算中"先乘除后加减、有括号先算括号"的灵活运用；用四则关系验算与求未知数。
易错点
①0÷0 误认为有商；②混合运算顺序错（同级从左到右遗忘）；③括号嵌套时算错层级；④求未知数时关系用反。
教学建议
①用"故事情境"区分四则含义（合、剩、倍、分）；②0÷a 用"0 个苹果分给 a 人"理解；③混合运算采用"画线法"标注运算顺序；④设计"我是小老师"互测求未知数。
5.1.2 运算定律与简便计算
学习目标
理解并运用加法交换律、结合律，乘法交换律、结合律、分配律；能进行简便计算；培养数感与巧算意识。
知识要点
加法交换律（a+b=b+a）、加法结合律（(a+b)+c=a+(b+c)）；乘法交换律、结合律；乘法分配律（a×(b+c)=a×b+a×c）；连减性质（a−b−c=a−(b+c)）；连除性质（a÷b÷c=a÷(b×c)）。
教学难点
乘法分配律的逆向运用（提取公因数）；连减、连除性质的灵活应用；何时该简算、何时不该。
易错点
①分配律漏乘一项（a×(b+c) 算成 a×b+c）；②分配律与结合律混淆；③"凑整"时改变运算值；④连减与连除用错。
教学建议
①用面积模型（长方形分块）直观演示分配律；②"凑整"意识培养，每天 3 道简算练习；③对比"分配律"和"结合律"的形似题；④逆向训练：25×99+25 = 25×(99+1)。
5.1.3 小数的意义与性质
学习目标
理解小数的意义（分母是 10、100、1000 的分数）；掌握小数的性质（末尾添 0 去 0 大小不变）；会比较小数大小；掌握小数点位置移动规律。
知识要点
小数的意义（0.1 = 1/10）；小数数位顺序表（十分位、百分位、千分位）；小数的性质（0.1=0.10=0.100）；小数点移动规律（右移一位扩大 10 倍，左移一位缩小到 1/10）；小数与单位换算。
教学难点
小数点移动引起小数大小变化的理解；单位换算中的进率与方向；小数性质"末尾 0"与"中间 0"的区分。
易错点
①小数点移动方向与倍数关系错；②位数不够时不会补 0；③"末尾 0"与"中间 0"混淆；④单位换算时进率错（米—厘米是 100 进率）。
教学建议
①用米尺/人民币演示小数（0.1 元=1 角）；②小数点移动用"小数点搬家"动画演示；③"末尾 0 vs 中间 0"用对比题强化：0.5=0.50≠0.05；④结合成都天气数据（温度、降水量）做小数读写练习。

领域二 · 图形与几何

5.2.1 三角形
学习目标
认识三角形的特征、分类；理解三角形内角和（180°）；理解三角形任意两边之和大于第三边；能按角和按边分类。
知识要点
三角形的定义、顶点、边、角；三角形的高与底；按角分类（锐角三角形、直角三角形、钝角三角形）；按边分类（不等边、等腰、等边）；三角形内角和；三角形两边之和 > 第三边。
教学难点
三角形高的画法（特别是钝角三角形外部高）；内角和的实验探究与说理；"两边之和 > 第三边"的判断（需三次比较）。
易错点
①钝角三角形高的画法错；②"两边之和 > 第三边"只比一次；③等腰与等边关系混淆；④给三个角判断能否组成三角形时算错和。
教学建议
①内角和用"撕角拼平角"实验 + 几何画板验证；②钝角三角形高用拓印或方格纸辅助；③"两边之和"用小棒实验，让学生发现规律；④结合建筑（屋顶桁架、桥梁）的三角形稳定性。
5.2.2 图形的运动（轴对称、平移）
学习目标
认识轴对称图形及其对称轴；能在方格纸上画出轴对称图形的另一半；认识平移，能在方格纸上画出平移后的图形。
知识要点
轴对称图形定义；常见图形的对称轴数量（正方形 4 条、等边三角形 3 条、长方形 2 条、圆无数条）；对称点的性质（到对称轴距离相等）；平移的方向与距离。
教学难点
对称轴条数的判断（如平行四边形无对称轴常被误判）；在方格纸上画对称图形（关键点—对应点—连线）；平移距离的判断（看一个点而不是看两条边之间的间隔）。
易错点
①平行四边形误认为有对称轴；②画对称图形时关键点遗漏；③平移距离数错格子；④"图形间距"误作平移距离。
教学建议
①用折纸 + 镜像演示轴对称；②"画对称图"分三步：找关键点 → 数距离 → 描对应点；③平移距离教学时强调"看一个点移动了几格"；④结合剪纸、蜀锦纹样等传统文化元素。

领域三 · 统计与概率

5.3.1 折线统计图
学习目标
认识折线统计图的特点；能根据数据画折线统计图；能从图中分析数据变化趋势并做预测。
知识要点
折线统计图的结构（标题、横轴、纵轴、单位、描点、连线）；与条形统计图的对比（折线突出"变化趋势"）；画图步骤；数据变化趋势的描述（上升/下降/平稳/波动）。
教学难点
描点的准确性（横纵坐标对应）；趋势的描述从"读图"到"预测"的过渡；从趋势中提取规律性结论。
易错点
①描点时横纵坐标对错；②忘记连线或连线顺序错；③"波动"误判为"上升/下降"；④预测脱离数据范围过度外推。
教学建议
①用成都月平均气温数据画图，本地化情境强；②"描点—连线—读图—预测"四步法；③对比条形图与折线图，体会各自优势；④预测环节强调"基于趋势的合理推断"。

领域四 · 综合与实践

5.4.1 营养午餐
学习目标
综合运用小数加减、统计、估算等知识；经历"调查—分析—设计—评价"的实践过程；培养数学应用与健康饮食意识。
知识要点
午餐营养标准（热量、蛋白质等）；菜谱热量与营养的简单计算；用统计图呈现分析结果；设计符合标准的午餐方案。
教学难点
数据收集的规范性；多约束条件下方案设计（热量达标 + 营养均衡 + 偏好）；结果的呈现与评价。
易错点
①热量/营养单位换算错；②只看单一指标忽略均衡；③方案设计脱离实际成本；④分析结论缺乏数据支撑。
教学建议
①结合学校食堂菜单做真实调研；②分组合作，每组设计一周午餐方案；③用 Excel/表格辅助计算，渗透信息技术；④邀请校医或家长营养师参与评价。
第六章

学年教学建议

学期规划、课时分配与单元衔接建议。

一、学期规划建议

根据成都市中小学 2025—2026 学年校历，上学期约 20 周、下学期约 20 周。建议如下分配：

学期周次主要内容建议课时
上册1-3大数的认识10
上册4-7三位数乘两位数 + 除数是两位数的除法16
上册8-10线段、直线、射线与角 + 平行与垂直12
上册11-13平行四边形与梯形 + 条形统计图10
上册14-151 亿有多大（综合实践）+ 期中复习6
上册16-20总复习 + 期末测评14
下册1-3四则运算的意义与关系10
下册4-6运算定律与简便计算10
下册7-9小数的意义与性质10
下册10-12三角形 + 图形的运动12
下册13-14折线统计图 + 营养午餐8
下册15-20总复习 + 期末测评16

二、课时分配原则

  • 新授:练习:复习 = 6:3:1：每个单元新授为主，配套练习，单元末复习
  • 周课时 4-5 节：依据《义务教育课程方案（2022 年版）》数学学科课时建议
  • 弹性预留：每月预留 1 课时处理学情反馈与差异化辅导
  • 综合实践：每学期至少 1 次，建议安排在月中（避免与期中期末冲突）

三、单元衔接建议

四年级上下册在知识脉络上有以下关键衔接点，班主任需特别关注：

  • 大数 → 小数：上册大数的认识 + 下册小数的意义，构成"数的扩展"完整脉络，可在下册开学初复习上册数位顺序表
  • 上册乘除 → 下册运算定律：上册笔算能力是下册简算的基础，建议寒假作业中保留 5 道/天的乘除练习
  • 上册图形特征 → 下册三角形：上册平行四边形与梯形是下册三角形的认知基础，可在下册开学用 1 课时回顾
  • 条形图 → 折线图：下册折线统计图前 1 课时复习条形图，对比引入

四、学情关注重点

四年级是小学数学"分化期"的开端，班主任应重点关注：

  1. 计算能力分化：除数两位数的除法是分水岭，对试商困难的学生提供"四舍五入试商表"辅助
  2. 抽象思维过渡：运算定律、小数意义等单元抽象度高，注意直观演示
  3. 学习习惯养成：草稿规范、验算意识、读题方法在本学年定型
  4. 差异化教学：建议每月做一次学情小测，分层辅导
第七章

评价建议

过程性评价、终结性评价与评价工具示例。

一、过程性评价（占比 40%）

过程性评价聚焦学生学习过程中的表现，建议采用多维度记录：

评价维度评价方式权重记录工具
课堂参与提问、回答、讨论、展示10%课堂观察记录表
作业完成及时性、正确率、订正态度10%作业批改记录
单元小测每单元结束 1 次，20-30 分钟10%小测分析表
综合实践活动参与度 + 成果质量10%实践评价量规

二、终结性评价（占比 60%）

终结性评价以学期为单位，包括期中与期末测评：

  • 期中测评：上册覆盖前 3 单元，下册覆盖前 2 单元；题型以基础题为主（占 70%），应用题 20%，拓展题 10%
  • 期末测评：全册范围；基础题 60%、应用题 25%、综合题 15%；时长 80 分钟
  • 测评命题原则：①覆盖课标核心知识点 80% 以上；②避免偏题怪题；③融入真实情境；④有梯度，区分度高

三、评价工具示例

建议班主任建立以下三类评价工具，便于系统追踪学生成长：

  • 学生成长档案袋：收集每月代表性作业、单元小测、实践成果，期末做成"我的数学成长"小册
  • 课堂观察量表：5 维度（专注、参与、表达、合作、创新），每周抽 5 名学生记录
  • 错题追踪表：按知识单元归类，记录每位学生的典型错题与订正情况

四、评价结果运用

评价不只是打分，更要"以评促学"。建议：

  1. 个体反馈：每月 1 次一对一面谈，基于数据指出进步与不足
  2. 群体分析：每次小测后做班级错题分布分析，针对性调整教学
  3. 家校沟通：每学期 2 次正式反馈（家长会 + 个性化报告），避免单纯分数通报
  4. 差异化辅导：按评价结果分层（A 巩固 / B 提升 / C 拓展），每周 1 次小班辅导
第八章

常见错题与对策

按知识点分类的错题案例与教学对策。

【错题 1 · 大数】读出 200040005
学生错误：二千万四千零五
教学对策：用数位顺序表逐位对照；强调"每级末尾的 0 不读，中间 0 要读"。正确读法：二亿零四万零五。
【错题 2 · 大数】把 3580000 改写成用"万"作单位的数
学生错误：358 万 = 3580000（混淆"改写"与"省略"）
教学对策：区分"改写"（精确值，用 =）与"省略"（近似值，用 ≈）。改写后应为 358 万；省略万位后是 358 万。
【错题 3 · 乘法】计算 240×50
学生错误：1200（漏补末尾 0）
教学对策：先算 24×5=120，再数两个因数末尾共有几个 0（共 2 个），所以 240×50=12000。强化"先算数字再补 0"。
【错题 4 · 除法】962÷26 的商
学生错误：商 27 余 20（试商 4 后未调商）
教学对策：用"四舍"试商 4，96÷26=3…18；继续 182÷26，试商 7，182-182=0。正确商 37。强化"试商—验商—定商"三步法。
【错题 5 · 角的度量】量一个 60° 角，读作 120°
学生错误：读错了刻度圈（读外圈而非内圈）
教学对策：量角前先看 0 刻度线对准哪一圈，对准内圈读内圈。强化"0 线—内/外—读数"三步。
【错题 6 · 平行四边形】判断"长方形是平行四边形"
学生错误：判断为"否"
教学对策：回归定义"两组对边分别平行的四边形"，长方形两组对边都平行，所以是平行四边形（特殊的）。用集合图强化包含关系。
【错题 7 · 运算定律】25×(40+4) 简算
学生错误：25×40+4 = 1004（漏乘一项）
教学对策：用面积模型演示"一个数乘两数之和 = 分别相乘再相加"。正确：25×40+25×4 = 1000+100 = 1100。
【错题 8 · 小数】0.5 与 0.50 的大小关系
学生错误：0.5 < 0.50（认为位数越多越大）
教学对策：小数性质——末尾添 0 大小不变，所以 0.5 = 0.50。强调"末尾 0"与"中间 0"区别：0.5 ≠ 0.05。
【错题 9 · 三角形】3 cm、4 cm、7 cm 三根小棒能否组成三角形
学生错误：能（只比较了 3+4 > 7）
教学对策：必须满足任意两边之和大于第三边，即三次比较都要满足。3+4=7，不大于 7，所以不能组成。
【错题 10 · 平移】图形向右平移 3 格后，两图形之间间隔了 3 格，问平移距离
学生错误：6 格（把图形间距当平移距离）
教学对策：平移距离 = 一个关键点移动的距离。引导学生标注一个顶点，数该顶点移动了 3 格。
