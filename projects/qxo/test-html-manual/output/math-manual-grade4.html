第一章

前言 · 写给四年级数学班主任

本手册的定位、使用方法与内容来源声明。

一、手册定位

本手册是面向成都市泡桐树小学四年级数学班主任的教学参考工具书，旨在为班主任提供一份系统化、可查询、可复用的学期教学速查资料。它不是教材的替代品，而是基于国家课程标准和公开教学大纲整理的教学组织参考，帮助班主任在学期规划、单元备课、错题分析、评价设计等环节快速定位信息。

二、使用方法

  • 学期初：参阅"第六章 学年教学建议"完成学期规划与课时分配
  • 单元备课：查阅"第四章 / 第五章"对应知识单元，获取学习目标、教学难点、易错点
  • 阶段性评价：参阅"第七章 评价建议"设计过程性评价工具
  • 错题复盘：参阅"第八章 常见错题与对策"针对性辅导
  • 家长沟通：可基于本手册内容向家长解释学习目标与评价方式

三、内容来源声明

⚠️ 内容依据
本手册内容依据以下公开资料整理：
1. 《义务教育数学课程标准（2022 年版）》—— 中华人民共和国教育部制定
2. 人教版小学数学四年级（上、下册）公开知识点大纲
3. 公开教学参考资料与教研成果

本手册不包含任何教材原文复制。教师在教学中应以最新版教材为准，本手册仅作为教学组织的参考。

四、特色说明

本手册在组织上有以下三个特点，便于班主任高效使用：

  1. 四大领域贯通：按"数与代数 / 图形与几何 / 统计与概率 / 综合与实践"四大领域组织，与新课标核心素养对接
  2. 五要素知识单元：每个知识单元含"学习目标 / 知识要点 / 教学难点 / 易错点 / 教学建议"五个要素，结构统一
  3. 本地化适配：结合泡桐树小学成都市区域背景，融入本地教学节奏与学情特点的考量

第三章

课程标准与核心素养

基于《义务教育数学课程标准（2022 年版）》，明确四年级数学的核心素养与课程目标。

一、数学核心素养（2022 课标）

根据 2022 版课标，小学数学核心素养主要包括以下三个方面，贯穿四年级教学全过程：

  • 会用数学的眼光观察现实世界：能从生活情境中抽象出数量关系与空间形式
  • 会用数学的思维思考现实世界：能运用归纳、演绎、类比等推理方式分析问题
  • 会用数学的语言表达现实世界：能用数学符号、图表、模型等表达和交流

二、第二学段（4-6 年级）课程目标

四年级属于第二学段，本学段在"数与代数 / 图形与几何 / 统计与概率 / 综合与实践"四大领域有如下总体要求：

  • 数与代数：扩展对自然数的认识，理解大数的意义；掌握三位数乘两位数、除数是两位数的除法；初步认识分数与含义；探索运算规律
  • 图形与几何：认识线段、直线、射线与角；理解平行与垂直；认识三角形、平行四边形、梯形的特征；体会图形的运动
  • 统计与概率：经历简单的数据收集、整理、描述和分析过程；认识条形统计图与折线统计图
  • 综合与实践：在真实情境中综合运用所学知识解决问题，体会数学与生活的联系

三、四年级教学重点

四年级是小学数学从"具体运算"向"抽象运算"过渡的关键年级，教学应重点关注：

  1. 从算术思维向代数思维的早期渗透（用字母表示数、等量关系）
  2. 空间观念的发展（从立体到平面、从直观到抽象）
  3. 数据处理意识的建立（从单一数据到多源数据的分析）
  4. 数学应用意识的培养（综合与实践领域的真实问题解决）
第四章

上册知识点详解

人教版四年级上册，按四大领域组织。每个知识单元含学习目标 / 知识要点 / 教学难点 / 易错点 / 教学建议五要素。

领域一 · 数与代数

4.1.1 大数的认识
学习目标
认识万级和亿级的数位、计数单位；掌握亿以内数的读法和写法；会用"四舍五入"法求近似数；体会大数在生活中的应用。
知识要点
数位顺序表（个位、十位…千亿位）；计数单位（一、十…千亿）；每四位一级（个级、万级、亿级）；多位数读写规则；用"万"或"亿"作单位表示大数。
教学难点
中间或末尾有 0 的多位数读法（如 20004000 读作"二千万四千"）；数位与计数单位的区分；亿以内数的写法（尤其 0 的占位）。
易错点
①读数时漏读"零"或多读"零"；②写数时数位对错；③"四舍五入"看错省略位；④把"="与"≈"用混。
教学建议
①用数位顺序表贯穿全单元，让学生随时对照；②结合成都本地数据（如人口、GDP）增强情境感；③设计"读—写—比"三步训练，每堂课 5 分钟巩固；④"四舍五入"借助数轴直观理解，避免死记口诀。
4.1.2 三位数乘两位数
学习目标
掌握三位数乘两位数的笔算方法；理解积的变化规律；能用所学知识解决简单的实际问题。
知识要点
三位数乘两位数的笔算步骤；因数中间或末尾有 0 的简算；积的变化规律（一个因数不变，另一个因数乘几，积也乘几）；常见数量关系：单价×数量=总价、速度×时间=路程。
教学难点
竖式书写时部分积的对位；积的变化规律的归纳与表述；进位时的连续处理。
易错点
①部分积的数位对错；②进位漏加；③末尾 0 的简算时少补 0；④"速度×时间=路程"中单位不统一。
教学建议
①先复习两位数乘两位数，迁移算法；②用方格图/面积模型直观解释部分积的含义；③对比"末尾有 0"和"中间有 0"两种情况，避免混淆；④结合生活实例（出行路程、采购总价）巩固数量关系。
4.1.3 除数是两位数的除法
学习目标
掌握除数是两位数的笔算除法；理解商的变化规律；能用四舍五入法试商、调商；解决简单实际问题。
知识要点
四舍五入法试商；调商的方法（四舍试商可能偏大需调小，五入试商可能偏小需调大）；商的变化规律（被除数和除数同时乘/除以相同的数，商不变）；有余数除法的验算。
教学难点
试商方法的灵活选择；调商时的判断；商不变规律中"余数变化"的理解。
易错点
①试商后忘记乘减直接写商；②调商后未重新计算；③商的数位定位错；④"商不变"中余数也认为不变。
教学建议
①设计"试商—调商"专项训练，每天 3 题；②用"估商—验商—定商"三步法口诀；③商不变规律用具体例题（24÷6、48÷12、96÷24）让学生观察归纳；④结合购物分摊、平均分配等情境出题。

领域二 · 图形与几何

4.2.1 线段、直线、射线与角
学习目标
认识线段、直线、射线的特征与区别；认识角的定义、表示方法；会用量角器量角、画指定度数的角。
知识要点
线段（两个端点、可度量）、直线（无端点、不可度量）、射线（一个端点、向一方无限延伸）；角的定义（从一点引出两条射线所组成的图形）；角的表示（∠1、∠ABC）；角的分类（锐角、直角、钝角、平角、周角）。
教学难点
三种线的区别与联系；射线方向性的理解；量角器的正确使用（中心对顶点、0 刻度线对一边）。
易错点
①直线与射线混淆；②量角时读错内/外圈刻度；③画角时方向错；④"平角 = 180°"与"直线"概念混淆。
教学建议
①用激光笔/手电筒演示射线；②量角器教学分两步：先对点对边、再读刻度；③画角采用"一对点、二对线、三找点、四连线"四步法；④结合折纸活动体会平角与周角。
4.2.2 平行与垂直
学习目标
理解同一平面内两条直线的位置关系（平行、相交、垂直）；会用直尺和三角尺画平行线与垂线；理解平行四边形和梯形的高。
知识要点
平行的定义（在同一平面内不相交）；垂直的定义（相交成直角）；平行与垂直的符号表示（∥、⊥）；点到直线的距离（垂线段最短）；画垂线和平行线的方法。
教学难点
"同一平面内"这一前提的理解；用三角尺画平行线/垂线的操作步骤；点到直线的距离与垂线段的关系。
易错点
①忽视"同一平面内"；②画垂线时三角尺放错位置；③把"垂直"等同于"竖直"；④平行四边形的高画到了顶点上。
教学建议
①用方格纸辅助理解平行；②"画平行线"用"平移三角尺"操作法，分步示范；③结合生活中的平行与垂直（斑马线、门框）增强直观；④画高时强调"从底边的对边任意一点向底边作垂线"。
4.2.3 平行四边形与梯形
学习目标
认识平行四边形和梯形的特征；理解它们与长方形、正方形的关系；会画平行四边形和梯形的高。
知识要点
平行四边形（两组对边分别平行）；梯形（只有一组对边平行）；平行四边形的底与高；梯形的上底、下底、高、腰；等腰梯形；四边形分类关系图（正方形 ⊂ 长方形 ⊂ 平行四边形 ⊂ 四边形）。
教学难点
四边形之间的包含关系；平行四边形"高"的多对应性（任一边都可作底）；梯形高的画法。
易错点
①梯形与平行四边形混淆（"只有一组"vs"两组"）；②高的画法错位；③认为"长方形不是平行四边形"；④梯形上底/下底颠倒。
教学建议
①用集合图直观展示四边形关系；②用四根木条做"平行四边形变形"演示，体会不稳定性；③"高"的教学采用"哪条边作底，就从对边向它作垂线"的统一表述；④结合成都街区的梯形屋顶、平行四边形地砖等实例。

领域三 · 统计与概率

4.3.1 条形统计图
学习目标
认识条形统计图（1 格表示多个单位）；能根据数据完成条形统计图；能从图中获取信息、分析数据。
知识要点
条形统计图的结构（标题、横轴、纵轴、单位、直条）；1 格表示多个单位的确定方法；制作条形统计图的步骤；从图中读数据、比较数据、做出简单判断。
教学难点
1 格表示多少单位的确定（依据数据范围与美观）；不完整数据的标注；分析数据时从"读数"到"判断"的过渡。
易错点
①1 格表示的单位错；②纵轴刻度不连续时未标"折线"；③直条高度对错刻度；④只读数不分析。
教学建议
①用班级真实数据（如身高、阅读量）作图；②"1 格表示几"让学生讨论决定，培养数据直觉；③分析环节用"我看到… 我想到… 我建议…"三步法；④对比表格与统计图，体会图形直观性。

领域四 · 综合与实践

4.4.1 1 亿有多大
学习目标
通过探究活动体会 1 亿的大小；经历"猜想—设计—测量—推算—得出结论"的研究过程；培养数感与估测能力。
知识要点
1 亿的数感建立（1 亿粒大米约多少千克？1 亿步约多少千米？）；推算方法（局部测量 → 按比例放大）；活动方案的设计与实施。
教学难点
由"小样本"推算"1 亿"的逻辑；活动方案的设计；估测结果的合理性判断。
易错点
①测量样本太小，误差过大；②推算时单位换算错；③结论脱离常识（如 1 亿粒米 > 一座山）；④活动记录不完整。
教学建议
①分组探究不同主题（大米、纸张、步数、硬币），增加多样性；②强调"先估、再测、后算"三步；③用成都到北京的距离（约 1800 km）作参照，让结果具象化；④活动后做"我的发现"分享。
